github操作遇到的几个问题

Nov 28, 2014

今天操作github在更新数据的时候报以下错误

有两种方式解决：

第一种 用强覆盖方式用你本地的代码替代git仓库内的内容,命令行执行如下代码：

$ git push -f

第二种 先把git的东西fetch到你本地然后merge后再push

$ git fetch

$ git merge

其实第二种方式的这两行代码等价于：

$ git pull
